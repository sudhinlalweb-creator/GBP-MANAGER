# Product Roadmap Baseline

## Phase 1

- Re-baseline repo structure into `frontend/` and `backend/`
- Establish SaaS primitives and environment contracts
- Prepare organization-first multi-tenant schema
- Replace single-app Docker setup with full-stack Compose

## Phase 2

- Implement Google Business Profile OAuth and sync foundations
- Add location and profile dashboard services
- Harden Celery workflows for sync jobs

## Phase 3

- Add AI audit engine, visibility scoring, and recommendation pipelines
- Introduce provider abstraction for OpenAI and Gemini workloads

## Phase 4

- Build keyword tracking, rank history, local grid heatmaps, and competitor data

## Phase 5

- Expand into reviews, Google posts, reports, automation, and agency workflows
